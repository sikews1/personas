Tradueix enunciats a equacions

Què és una equació de primer grau?

Una equació de primer grau amb una incògnita és una igualtat on la variable (x) apareix elevada a l'exponent 1. Forma general:

a·x + b = 0

amb a ≠ 0. Resoldre l'equació vol dir trobar el valor de x que fa certa la igualtat.

Passos per resoldre-la

  1. Treure parèntesis (propietat distributiva).
  2. Treure denominadors multiplicant pel m.c.m.
  3. Agrupar termes amb x a un costat i els nombres a l'altre.
  4. Reduir termes semblants.
  5. Aïllar la x dividint pel coeficient.

Exemple resolt

Resolem: 3x + 5 = 2x + 9

1) 3x − 2x + 5 = 9
2) 3x − 2x = 9 − 5
3) x = 4

Què és una equació de segon grau?

És una equació de la forma:

a·x² + b·x + c = 0

amb a ≠ 0. Pot tenir dues solucions, una de doble, o cap solució real.

Fórmula general

x = ( −b ± √(b² − 4ac) ) / (2a)

Δ = b² − 4ac (discriminant):

  • Δ > 0 → dues solucions reals.
  • Δ = 0 → solució doble.
  • Δ < 0 → cap solució real.

Exemple resolt

Resolem: x² − 5x + 6 = 0

a = 1, b = −5, c = 6
Δ = 25 − 24 = 1
x = (5 ± 1) / 2
x₁ = 3, x₂ = 2

Què és un sistema d'equacions?

Dues equacions de primer grau que han de ser certes alhora:

a₁x + b₁y = c₁
a₂x + b₂y = c₂

Cal trobar x i y que compleixen les dues equacions.

Mètodes de resolució

  • Substitució: aïllar una variable i substituir-la a l'altra equació.
  • Igualació: aïllar la mateixa variable a les dues i igualar.
  • Reducció: sumar o restar les equacions per eliminar una variable.

Exemple resolt (reducció)

x + y = 7
x − y = 3
Sumem: 2x = 10 → x = 5
Substituïm: 5 + y = 7 → y = 2

Problemes d'enunciat

Tradueix l'enunciat a una equació seguint aquests passos:

  1. Llegeix l'enunciat amb atenció.
  2. Assigna x a la incògnita.
  3. Expressa la resta de dades en funció de x.
  4. Planteja l'equació.
  5. Resol i comprova.

Exemple

"La suma de dos nombres consecutius és 27. Quins són?"

x + (x + 1) = 27
2x = 26 → x = 13
Els nombres són 13 i 14.
