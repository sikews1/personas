Què és una equació de primer grau?

Una equació de primer grau amb una incògnita és una igualtat on la variable (normalment x) apareix elevada a l'exponent 1. Forma general:

a·x + b = 0

on a ≠ 0. Resoldre l'equació vol dir trobar el valor de x que fa certa la igualtat.

Passos per resoldre-la

  1. Treure parèntesis (propietat distributiva).
  2. Treure denominadors multiplicant pel m.c.m.
  3. Agrupar termes amb x a un costat i els nombres a l'altre.
  4. Reduir termes semblants.
  5. Aïllar la x dividint pel coeficient.

Exemple resolt

Resolem: 3x + 5 = 2x + 9

1) 3x − 2x + 5 = 9
2) 3x − 2x = 9 − 5
3) x = 4

Què és una equació de segon grau?

És una equació de la forma:

a·x² + b·x + c = 0

amb a ≠ 0. Pot tenir dues solucions, una de sola (doble) o cap solució real.

Fórmula general

x = ( −b ± √(b² − 4ac) ) / (2a)

L'expressió Δ = b² − 4ac es diu discriminant:

  • Si Δ > 0: dues solucions reals diferents.
  • Si Δ = 0: una solució doble.
  • Si Δ < 0: cap solució real.

Exemple resolt

Resolem: x² − 5x + 6 = 0

a = 1, b = −5, c = 6
Δ = (−5)² − 4·1·6 = 25 − 24 = 1
x = (5 ± √1) / 2 = (5 ± 1) / 2
x₁ = 3, x₂ = 2

Què és un sistema d'equacions?

Un sistema de dues equacions amb dues incògnites són dues equacions de primer grau que han de ser certes alhora:

a₁x + b₁y = c₁
a₂x + b₂y = c₂

Resoldre'l vol dir trobar els valors de x i y que compleixen les dues equacions.

Mètodes de resolució

  • Substitució: aïllar una variable en una equació i substituir-la a l'altra.
  • Igualació: aïllar la mateixa variable a les dues equacions i igualar.
  • Reducció: sumar o restar les equacions per eliminar una variable.

Exemple resolt (reducció)

Resolem:

x + y = 7
x − y = 3
Sumem: 2x = 10 → x = 5
Substituïm a la primera: 5 + y = 7 → y = 2
Solució: x = 5, y = 2

Problemes d'enunciat

Els problemes d'enunciat es resolen traduint el text a una equació. Segueix aquests passos:

  1. Llegeix l'enunciat amb atenció.
  2. Assigna una variable (x) a la incògnita.
  3. Expressa la resta de dades en funció de x.
  4. Planteja l'equació a partir de la condició del problema.
  5. Resol l'equació.
  6. Comprova que la solució té sentit.

Exemple

"La suma de dos nombres consecutius és 27. Quins són?"

Sigui x el primer nombre. L'altre és x + 1.
x + (x + 1) = 27
2x + 1 = 27 → 2x = 26 → x = 13
Els nombres són 13 i 14.
